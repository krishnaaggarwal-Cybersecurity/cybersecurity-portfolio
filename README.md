## Repository Contents

| File | Description |
|---|---|
| `caesar_cipher.c` | Substitution cipher — modular arithmetic, key-based shift, why it breaks |
| `vigenere_cipher.c` | Polyalphabetic cipher — keyword scheduling, defeats frequency analysis, still breakable by Kasiski |
| `xor_cipher.c` | Bitwise XOR encryption — foundation of stream ciphers and OTP, hex I/O |
| `password_strength.c` | Entropy-based password evaluator — charset analysis, pattern detection, scoring |

### Why these projects?

Each tool is chosen to build genuine understanding, not to impress with complexity:

- **Caesar → Vigenère**: Shows how adding key complexity improves security, and why it's still not enough
- **XOR Cipher**: The actual bitwise operation inside AES, ChaCha20, and every modern cipher — understanding this is understanding the foundation
- **Password Checker**: A practical security tool with real-world application. Demonstrates thinking about entropy, not just "has a capital letter"
