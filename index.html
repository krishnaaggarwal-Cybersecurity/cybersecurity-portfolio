Germicidal Automatic Washing Machine

EMC 2025 · Team Technophile (Leader) · 6-Member Team

Enhanced version of the INSPIRE project. Added UV-C germicidal irradiation (180–280nm wavelength) to kill bacteria and pathogens during the wash cycle. Integrated DC motor control, submersible pump, centrifugal spin mechanism, and toggle-switch control panel. Demonstrates applied physics, embedded control logic, and the security-adjacent question of system safety: ensuring UV-C operates only when the drum is closed — a physical access control problem.

Caesar Cipher — Encryption / Decryption Tool

C Language · Cryptography Fundamentals · GitHub

Command-line Caesar Cipher encoder/decoder in C. Handles uppercase and lowercase ASCII, preserves non-alphabetic characters, implements modular arithmetic correctly (key 26 = identity). A deliberately simple project — its purpose is to understand why substitution ciphers are broken, not to claim it's secure. Companion projects include Vigenère Cipher and XOR Cipher, building toward understanding of stream vs block cipher logic.

C Language Symmetric Encryption Modular Arithmetic Cryptanalysis Context

Vigenère Cipher

C Language · Polyalphabetic Substitution · GitHub

Polyalphabetic cipher that uses a repeating keyword to shift characters — defeating simple frequency analysis that breaks Caesar. Written in C, handles variable-length keys, processes only alpha characters. Demonstrates understanding of how adding key complexity increases cryptographic strength, and why even this is broken by Kasiski examination — building real intuition about what "security" means in cryptography.

C Language Polyalphabetic Cipher Key Scheduling Cryptanalysis Awareness
View Code

XOR Cipher & Password Strength Checker

C Language · Bitwise Operations · Security Tool

XOR cipher demonstrating the fundamental bitwise operation behind modern stream ciphers and OTP encryption — one-time pad security when the key is truly random. Password strength checker evaluates entropy, character set diversity, common pattern detection, and provides actionable feedback. Both tools are practical demonstrations of security concepts rather than toys.

Statement of Purpose

I am Krishna Aggarwal, a Class 12 student from Sarvodaya Bal Vidyalaya, J&K Block, North-East Delhi, applying for the B.Cybersecurity program at IIT Kanpur. My interest in security is not theoretical — it grew from building systems and understanding, physically, what it means for something to fail.

In Class 9, I designed a portable automatic washing machine from first principles: a wooden chassis, submersible pump, DC-to-AC converter, and electric motor. No kit. No instructions. The project was selected by the Department of Science & Technology, Government of India for the INSPIRE MANAK Award — a ₹10,000 cash prize — and was later exhibited at the District and State Level under the GNCT Delhi Science Branch. The lesson I took from that project was not that I had built something impressive. It was that every system has a failure mode, and the person who finds it first wins.

That mindset is why cybersecurity draws me. I completed a Certificate in C and C++ Programming with Grade A from ICE Computer Education, and began writing small cryptography tools — Caesar cipher, Vigenère cipher, XOR cipher, and a password strength checker — not because they are sophisticated, but because implementing them forces you to understand exactly why they are weak. I am currently pursuing a DCAP diploma from IICS Gokulpuri, covering networking fundamentals and operating systems concepts. In 2025, I led a 6-member team at the Engineering Model Competition with an enhanced germicidal version of my INSPIRE project.

I am not applying to IIT Kanpur because I have already mastered cybersecurity. I am applying because I have demonstrated, repeatedly, that I learn by building things — and I want to build things that matter. India's digital infrastructure expands daily into territory that is poorly defended. I want to be one of the people who changes that, starting with the rigorous, foundational education that IIT Kanpur's B.Cybersecurity program offers.

I am prepared to work harder than anyone in the program. I ask for the chance to prove it.
